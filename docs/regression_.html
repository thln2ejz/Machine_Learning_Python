Linear Regression¶

In this section, we consider simple linear regression which consists of fitting a linear function to the data. The main assumption here is that the data comes from a linear model given by:

$$y = \beta_0 + \beta_1 x + \epsilon$$

Thereafter, we try to estimate the model with the regression line:

$$\widehat{y} = \beta_0 + \beta_1 x$$

The parameters are usually estimated using the least squares method which minimizes the sum of squares difference between the estimated values and the actual observed data $[y_1,y_2,...,y_n]$:

$$J_{LS} = \sum_i \epsilon_i^2 = \sum_i (y_i -\beta_0-\beta_1 x_i)^2$$

The parameter estimates may be obtained analytically by setting the partial derivatives with respect to the offset and the other coefficients of the above equation to zero and solving to arrive at:

$$\widehat{\beta} = (\textbf{X}^T \textbf{X})^{-1} \textbf{X}^T \textbf{y} $$

where $\textbf{X} = \pmatrix{ x_1 & 1 \\ x_2 & 1 \\ . & . \\ . & . \\ x_n & 1 \\ }$, $\textbf{y} = \pmatrix{ y_1 \\ y_2 \\ . \\ . \\ y_n }$, and $\widehat{\beta} = \pmatrix{ \beta_1 \\ \beta_0 }$

def linear_regression(X, y):
    X = np.stack([X,np.ones(X.shape[0])],axis=1)
    beta = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)
    return beta

Regression Quality¶

It was mentioned in the lecture notes that beginning from the identity:

$$ y_i - \widehat{y}_i = (y_i - \overline{y}_i) - (\widehat{y}_i - \overline{y}_i)$$

the following may be derived:

$$\sum_i (y_i - \overline{y}_i)^2 = \sum_i (\widehat{y}_i - \overline{y}_i)^2 + \sum_i (y_i - \widehat{y}_i)^2 $$

We may then assess the quality of regression by evaluating the ratio $R^2$ consisting of dividing the first RHS term in the above equation by the LHS term. A ratio close to 1 is indicative of a linear trend or non-zero slope in the data.

Nonlinear Regression¶

In this section, we explore kernel regression as a form of local weighted regression.

Exercise 4. The function kernel_regression() is provided below. The function takes as input the training set abscissa values X_train and ordinate values y_train, a test set input X_test, and kernel width h. The output of the function is a set of regression predictions y_pred. For every test object or query point q, kernel_regression() gives a kernel smoothed output by weighing training set objects with a kernel function K() as follows:

$$\widehat{y}(q) = \dfrac{\sum_i K \left( \dfrac{x_i-q}{h} \right) y_i}{\sum_i K \left( \dfrac{x_i-q}{h} \right)}$$

The kernel function employed by kernel_regression() is a Gaussian kernel with kernel width h representing the standard deviation.

In [6]:
def kernel_regression(X_train, y_train, X_test, h): #1D case
    y_pred = []
    for i in range(len(X_test)):
        K = np.exp(-(X_train-X_test[i])**2/h**2)
        y_pred.append((K*y_train).sum()/np.sum(K)) 
    return y_pred

def optimize_h(X, y, h):  #1D case
    from sklearn.model_selection import train_test_split
    rep = 5
    mse = np.zeros([rep,len(h)])
    for j, bandwidth in enumerate(h):
        for i in range(5):
            X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)
            y_pred = kernel_regression(X_train, y_train, X_test, bandwidth)
            mse[i,j] = ((y_pred - y_test)**2).mean() 
    plt.errorbar(h, mse.mean(axis=0), mse.std(axis=0), linewidth=2)
    plt.xlabel('h')
    plt.ylabel('mse')
    plt.show()
